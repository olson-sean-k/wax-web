pub const fn is_contextual_meta_character(x: char) -> bool
Expand description

Returns true if the given character is a contextual meta-character.

Contextual meta-characters may only be escaped in particular contexts, such as hyphens - in character class expressions. Elsewhere, they are interpreted as literals. To detect non-contextual meta-characters, use is_meta_character.
